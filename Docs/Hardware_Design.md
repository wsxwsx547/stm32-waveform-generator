# 硬件设计文档

## 系统架构

```
STM32F103C8T6 (72MHz)
├── DAC (PA4) ──→ 低通滤波器 ──→ 波形输出
├── TIM6 ──→ DAC触发 (TRGO)
├── DMA2_CH4 ──→ DAC数据搬运
├── USART1 (PA9/PA10) ──→ USB-TTL调试
├── GPIO ──→ 按键输入 (PB0/PB1/PB10)
├── GPIO ──→ 触摸检测 (PA0, TTP223)
├── I2C ──→ OLED显示 (PB6/PB7, SSD1306)
└── 电源: 5V USB → 3.3V LDO
```

## 引脚分配

| 引脚 | 功能 | 说明 |
|------|------|------|
| PA0 | 触摸输入 | TTP223信号，浮空输入 |
| PA4 | DAC输出 | 12位DAC通道1 |
| PA9 | USART1_TX | 调试串口发送 |
| PA10 | USART1_RX | 调试串口接收 |
| PB0 | 按键1 | 波形切换，下拉输入 |
| PB1 | 按键2 | 频率增加，下拉输入 |
| PB6 | I2C_SCL | OLED时钟线 |
| PB7 | I2C_SDA | OLED数据线 |
| PB10 | 按键3 | 频率减小，下拉输入 |

## 电路设计要点

1. **电源设计**: USB 5V经AMS1117-3.3降压至3.3V，MCU及DAC供电。每路电源就近放置100nF去耦电容。
2. **DAC输出**: PA4直连LM358运放做缓冲，输出阻抗低、驱动能力强。低通滤波器截止频率设为200kHz。
3. **触摸检测**: TTP223模块自研外围电路，灵敏度通过并联电容调节（0~50pF）。
4. **晶振电路**: 8MHz HSE配20pF负载电容，走线尽量短且等长。
5. **复位电路**: 10kΩ上拉+100nF电容+按键，确保上电可靠复位。

## PCB设计注意事项

- 4层板设计: Signal-GND-Power-Signal
- DAC走线远离数字信号，避免串扰
- 晶振走线差分等长，包地处理
- 触摸电极区域保持净空，避免寄生电容
